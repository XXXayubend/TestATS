# Gestion des Candidats et Offres - API REST Spring Boot

## Description

Mini systeme ATS  :
- **Candidats** : création et consultation de profils (nom, email, compétences, expérience, CV)
- **Offres** : création et consultation d'offres (titre, compétences requises)

## Fonctionnalités

- ✅ Création d'un candidat
- ✅ Liste de tous les candidats
- ✅ Création d'une offre 
- ✅ Liste de toutes les offres
- ✅ Evaluatoin du candidat avec un score
- ✅ Tests d'intégration des endpoints (Swagger)

## Technologies utilisées

- Spring Boot
- Spring Data JPA (Hibernate)
- Base de données MySQL
- OpenRouter
- Swagger
